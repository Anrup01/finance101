
# finance101

### requirements
- Python 3.6.9
- Flask 1.1.2
- Werkzeug 1.0.1

### install (https://flask.palletsprojects.com/en/1.1.x/installation/)
- fork repo
- cd finance101
- python3 -m venv venv 
- . venv/bin/activate
- sudo pip3 install Flask
